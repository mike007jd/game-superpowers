# Community Benchmarks

Game Superpowers should be tested in the open.

This repository welcomes same-task comparisons that help show where the workflow helps, where it does not, and how results change across models or hosts.

## What makes a useful benchmark

- same model for both runs
- same general task
- same output format
- only one intended workflow difference
- screenshots or video
- runnable or inspectable artifacts when possible

## Strong benchmark patterns

- raw prompt vs `/using-game-superpowers`
- model A vs model B with the same Game Superpowers flow
- Game Superpowers vs another game-focused agent workflow
- first-prompt comparison before any manual cleanup

## Recommended evidence

- prompt screenshots
- model name and runtime
- before / after screenshots
- downloadable artifacts such as single-file HTML outputs
- a short table describing what changed

## Suggested comparison dimensions

- bootability
- clarity of start flow
- input and gameplay feedback
- UI / HUD readability
- visual direction
- language consistency
- audio and effects
- whether the result is actually playable

## Good faith note

Not every benchmark will favor Game Superpowers.

That is fine. Useful community benchmarks are honest, reproducible enough to inspect, and explicit about what changed between runs.
